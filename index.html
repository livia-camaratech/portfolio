Projetos

Sistema de Controle de Horas de Alunos

HTML JavaScript CSS

Sistema web desenvolvido para resolver um problema real de controle de horas de trabalho, permitindo consulta individualizada por aluno com foco em privacidade, organização de dados e experiência do usuário. A aplicação exibe informações como carga horária pendente, dias de ausência e dados acadêmicos, com feedback visual em cores (verde e vermelho) para facilitar a interpretação.

Destaques:
  • Interface pensada para fácil visualização
  • Projeto criado para atender uma demanda real do ambiente de trabalho
  • Foco em organização de informações e experiência do usuário
